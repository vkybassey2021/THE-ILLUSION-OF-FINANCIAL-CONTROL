# The Illusion of Financial Control

This repository contains the code for the Data Science Lab project:

**"The Illusion of Financial Control: Literacy, Overconfidence and Financial Vulnerability in Italy"**

## Data
The analysis uses the 2023 Bank of Italy IACOFI survey.

The dataset is publicly available from the Bank of Italy website and is not included in this repository.

## Methodology
The project includes:
- Data cleaning and preparation
- Construction of financial capability and vulnerability indices
- Correlation analysis
- Linear regression models
- Interaction model
- Overconfidence (mismatch) analysis

## How to run
1. Open the R script or R Markdown file
2. Install required packages:
   - tidyverse
   - ggplot2
3. Run the code step by step
